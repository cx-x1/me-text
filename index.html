JSON API تخزين وتعديل المحتوى برمز

🔑 قم بإنشاء "كود" خاص بك، احفظ المحتوى، ثم استخدم الرابط ?api=json&code=كودك للحصول على JSON خالص.

أحرف إنجليزية، أرقام، _ و - فقط. يمكنك استخدامه لاحقاً لتعديل المحتوى.

📡 روابط API (تعرض JSON فقط بدون HTML):

ادخل كود واضغط حفظ لعرض الرابط

⚙️ كيف يعمل التعديل لاحقاً؟

  • 🔹 إنشاء / تعديل: أدخل نفس الكود الذي استخدمته سابقاً ثم غيّر المحتوى واضغط "حفظ" — سيتم تحديث البيانات.
  • 🔹 استرجاع JSON النقي: استخدم الرابط ?api=json&code=YOUR_CODE — هذه الصفحة تعرض JSON فقط بدون أي علامات HTML، مناسبة لأي تطبيق خارجي.
  • 🔹 استرجاع نص خام: ?api=raw&code=YOUR_CODE يعرض المحتوى كـ plain text.
  • 🔹 مثال لاستدعاء خارجي (Python, JS, PHP): fetch("https://yourdomain.com/page.html?api=json&code=my-config") → يحصل على JSON فوري.
